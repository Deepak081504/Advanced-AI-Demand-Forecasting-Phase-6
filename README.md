# Multi-Vendor E-Commerce Platform Backend

## Project Overview

This project is a Multi-Vendor E-Commerce Backend developed using FastAPI and PostgreSQL.

The system supports three types of users:

- Admin
- Vendor
- Customer

The platform allows vendors to manage products, customers to place orders, and administrators to monitor the entire system.

---

## Features

### Authentication

- User Registration
- User Login
- JWT Authentication
- Password Hashing using Bcrypt

### Role Management

- Admin Role
- Vendor Role
- Customer Role

### Product Management

- Create Product
- Update Product
- Delete Product
- View Products

### Cart Management

- Add to Cart
- Update Cart
- Remove from Cart
- View Cart

### Order Management

- Place Order
- View Orders
- Order Status Tracking

### Vendor Features

- Manage Products
- View Vendor Orders
- Update Order Status

### Admin Features

- Manage Users
- Manage Vendors
- View All Orders
- Monitor Platform Activity
